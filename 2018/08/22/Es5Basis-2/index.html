表达式和运算符

原始表达式

true //返回布尔值 真 false //返回布尔值 假 null //返回值 空 this //返回当前对象

在一个方法体内，this返回调用这个方法的对象。

函数表达式

函数表达式可以称为‘函数直接量’，

var square = function(){ return x*x; }

+ 运算符

1 + 2 = 3; '1' + '2' = '12'; 'hello' + 1 = 'hello1'; '12' + 1 = '121'

in运算符

var data = [7,8,9]; '0' in data //true 1 in data //true 3 in data //false

这里 in 前面的值，是键

instanceof运算符

左操作数是一个对象，右操作数标识对象的类。

如果左侧对象是右侧类的实例，则返回true

所有对象都是object的实例。

为了计算 0 in f js首先计算f.prototype,然后在原型链中查找o，如果找到，那么o是f的实例。

如果 f.prototype不在o的原型链中，那么o不是f的实例。

表达式计算

jiavascript可以解释运行由javascript源码组成的字符串，并产生一个值。通过全局函数eval()

eval("3+2") //5

eval 只有一个参数，如果传入的不是字符串，将直接返回这个参数，如果参数是字符串，会进行编译，编译失败会抛出错误。

eval使用了调用它的变量作用域环境。

全局eval()

delete

删除对象属性或者数组元素。他是同来进行删除操作，并不是用来返回一个值的。

var 0 = { x:1,y:2 } delete o.x; 'x' in o //false

var a = [1,2,3]; delete a[2]; 2 in a ; //fasle 元素2在数组中已经不存在了 a.length // 3 数组原来的长度不会改变

用var声明的变量不能删除，通过函数function语句定义函数和函数参数也不能删除。

void

浏览器不显示表达式结果，经常在html的a标签里看到。

语句

声明表达式

var

如果var语句没有指定初始化表达式，那么这个变量的初始值为undefined，变量在声明它们的函数或者脚本中都是有定义的，变量声明语句会被提前至脚本或者函数的顶部，但是初始化还是在原来的位置，在声明语句之前，变量值为undefined。

function

function f(x) {return x*x } //函数会被提前到所在作用域最顶端。

for

for(initialize;test;increment)
stament

初始化;循环条件判断;计数器更新

for循环便利链表结构

function tail(o){ for(;o.next;o=o.next)//根据判断o.next是不是真值执行遍历 return o ; //返回第一个不包含next属性的对象 }
